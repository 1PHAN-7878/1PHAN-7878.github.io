hexo无法显示图片的问题解决

2023-11-04

最近发现在上传一些包含图片的md文件时不能正常的显示，于是去寻找解决办法。 md文件中插入图片的语法为![]() 总体来说就是路径不对，无法找到正确的路径，于是更改默认的路径例如 我在source文件夹下建立了images的文件夹来存储图片，但是每次都要这样写一下路径不，不是嘞。 按照typora的偏好设置，可以让他为我们自己配置这样的路径。 如此一来在执行 hexo generate hexo deploy 即可发现成功显示图片
